## 시스템 요구사항

- Node 20 LTS 이상
- pnpm 10.2.1 이상

## 요구사항 설치

```bash
$ npm install -g pnpm@latest-10
```

## 실행

### 터미널에서 실행시

```bash
$ pnpm i
$ pnpm dev
```
